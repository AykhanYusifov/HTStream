Installation

Install Prerequisites

Make sure you have Boost libraries installed - version must be greater than 1.56.0

Additionally, make sure you have Cmake installed - the version must be greater than 2.8

Install Prerequisites for Linux

sudo apt-get upgrade
sudo apt-get update
sudo apt-get install cmake # Cmake install - Additional help here
sudo apt-get install libboost-all-dev # Boost install - Additional help here
rehash #might be needed if you are using zsh

Install Prerequisites for Mac

#Homebrew is required for the following instructions - Here is a tutorial to install homebrewbrew upgrade
brew install cmake # Cmake install - Additional help here
brew install boost # Boost install - Additional help here
rehash #might be needed if you are using zsh

Installation Guide


The basic install

git clone https://github.com/ibest/HTStream.git
cd HTStream
mkdir build && cd build
cmake ..
make #install to /usr/local/bin
rehash #might be needed if you are using zsh

All applications have unit test that can be run
