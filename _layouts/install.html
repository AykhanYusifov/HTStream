Installation

Install Prerequisites

Additionally, make sure you have Cmake installed - the version must be greater than 3.2

Install Prerequisites for Linux

sudo apt-get install cmake # Cmake install - Additional help here
sudo apt-get update && sudo apt-get upgrade
rehash #might be needed if you are using zsh

Make sure you have Boost libraries installed - version must be greater than 1.56.0

Install Prerequisites for Linux (sudo)

wget -O 1_60.tar.bz2 https://sourceforge.net/projects/boost/files/boost/1.60.0/boost_1_60_0.tar.bz2/download
tar --bzip2 -xf 1_60.tar.bz2
cd boost_1_60_0
./bootstrap.sh
sudo ./b2 install
rehash

Install Prerequisites for Linux (non sudo)

wget -O 1_60.tar.bz2 https://sourceforge.net/projects/boost/files/boost/1.60.0/boost_1_60_0.tar.bz2/download
tar --bzip2 -xf 1_60.tar.bz2
cd boost_1_60_0
./bootstrap.sh
./b2
BOOST_INCLUDE=$(pwd)
BOOST_INCLUDE_LIB=$(pwd)/stage/lib
#Need to modify the cmake command in install
#cmake -DCMAKE_INLCUDE_PATH=$BOOST_INCLUDE -DCMAKE_LIBRARY_PATH=$BOOST_INCLUDE_LIB ..
rehash

Install Prerequisites for Mac

#Homebrew is required for the following instructions - Here is a tutorial to install homebrewbrew upgrade
brew install cmake # Cmake install - Additional help here
brew install boost # Boost install - Additional help here
rehash #might be needed if you are using zsh

The basic install

git clone https://github.com/ibest/HTStream.git
cd HTStream
mkdir build && cd build
cmake ..
#with non sudo access please run this command
#cmake -DCMAKE_INCLUDE_PATH=$BOOST_INCLUDE -DCMAKE_LIBRARY_PATH=$BOOST_INCLUDE_LIB -DCMAKE_INSTALL_PREFIX=[PATH FOR EXECUTABLES TO BE WRITTEN TO] .. 
#PATH:=$PATH:[PATH FOR EXECUTABLES TO BE WRITTEN TO]/bin
make #install to /usr/local/bin with sudo access (or location specified with not sudo access)
rehash #might be needed if you are using zsh
